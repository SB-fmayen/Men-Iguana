# **App Name**: Iguana Menu

## Core Features:

- Menu Display: Display menu items in a card format, separated by categories.
- Category Sections: Organize the menu into distinct sections for each food category.
- Responsive Design: Ensure the menu is responsive and viewable on both desktop and mobile devices.
- Menu Item Details: Each menu item card should display name, price, and available options or extras.
- QR Code Access: Generate a QR code that directly links to the digital menu.

## Style Guidelines:

- Primary color: Vivid lime green (#C0FF33), reflecting the 'iguana' theme and providing a lively, attention-grabbing base.
- Background color: Very dark, desaturated green (#FFFFFF) for a contrasting and modern backdrop, letting the primary green 'pop'.
- Accent color: Black (#000000) to emphasize crucial details and CTA elements.
- Body and headline font: 'Bold Italic', a grotesque-style sans-serif, to provide a neutral and easily-readable experience.
- Card-based layout with clear separation between categories to ensure easy navigation.
- Use a responsive grid system to adapt to different screen sizes.
- Use minimalistic icons to represent food categories.
